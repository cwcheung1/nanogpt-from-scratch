A parameter is one number in a grid

A Linear layer connecting 4 inputs to 8 outputs needs one learnable number for every input–output pair — a 4×8 grid of connections. Every cell you see below is one parameter.

Inference, not training

How much memory to remember what's already been read?

Generating text one character at a time, the model would normally recompute every earlier position's key & value vectors on every single step — the exact waste lesson 2's bigram model flagged. Real serving avoids that: compute each position's key/value once, cache it, and just append as new tokens arrive.

Drag the sliders. The bars and the number below update live — watch how directly memory scales with context length and concurrent users.
